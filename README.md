# aitrack.work

A minimal task-based time tracker.

Built using Vue3 and [Vite.js](https://github.com/vitejs/vite), deployed via Netlify.

## Run @ `localhost`

1. `npm install`
2. `npm run dev`
3. Open `localhost:3000`
